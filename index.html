资本预算技术 (Capital Budgeting Techniques)

【核心定义】
• 资本预算：评估和选择长期投资的过程，与企业所有者财富最大化目标一致
• 资本支出：企业预期在超过1年的时间内产生效益的资金支出

【现金流组成（三个主要组成部分）】
• 初始投资：第0期项目的相关现金流出
• 营运现金流入：项目生命期内实施产生的增量税后现金流入
• 终值现金流：项目最后一年发生的税后非营运现金流

【初始投资计算公式】
初始投资 = 新资产安装成本 - 旧资产出售后税后收入 ± 净营运资本变化
其中：安装成本 = 新资产成本 + 安装费用 \$
【重要概念】
• 沉没成本：已经发生的现金支出，不影响当前决策的相关现金流
• 机会成本：拥有资产最佳替代用途的现金流，应作为现金流出包含
• 增量现金流：实施拟议资本支出预期产生的额外现金流

【营运现金流入计算】
税后营运现金流入 = 收入 - 费用（不含折旧和利息）
增量现金流入 = 新项目现金流入 - 现有项目现金流入 \$
\$
基础信息分析：
• 旧机器原价：80,000美元，预计5年使用寿命，残值10,000美元
• 年折旧额 = (80,000 - 10,000) ÷ 5 = 14,000美元
• 已使用2年，账面价值 = 80,000 - (14,000 × 2) = 52,000美元
• 当前市场价值：35,000美元
• 剩余使用寿命：3年 \$

• 年销售额：40,000美元
• 年运营成本：12,000美元
• 年净现金流 = 40,000 - 12,000 = 28,000美元
• 第3年末残值：10,000美元

• 第2年现值 = 28,000 ÷ (1.10)² = 23,140.50美元
• 第3年现值 = 38,000 ÷ (1.10)³ = 28,549.36美元 \$
继续使用旧机器的NPV = -35,000 + 25,454.55 + 23,140.50 + 28,549.36 = $42,144.41 \$
\$

• 安装费：10,000美元
• 净营运资本增加：12,000美元

资本预算技术 (Capital Budgeting Techniques)

【核心概念】
• 资本预算：评估和选择长期投资的过程
• 增量现金流：项目实施后的额外现金流
• 净现值（NPV）：项目未来现金流的现值减去初始投资

【现金流计算方法】
• 现金流入：销售收入等经营活动产生的现金收入
• 现金流出：经营费用、初始投资等现金支出
• 净现金流：现金流入减去现金流出

【净现值（NPV）计算公式】
$$NPV = \sum_{t=1}^{n} \frac{CF_t}{(1+r)^t} - \text{初始投资}$$
其中：$$CF_t$$ = 第t期现金流，$$r$$ = 折现率

【沉没成本与机会成本】
• 沉没成本：已发生的历史成本（如研发费用），不影响决策
• 机会成本：放弃最佳替代方案的成本，应计入现金流

【内部收益率（IRR）】
• 定义：使NPV=0的折现率
• 决策准则：IRR > 资本成本时接受项目
$$NPV = 0 = \sum_{t=1}^{n} \frac{CF_t}{(1+IRR)^t} - \text{初始投资}$$
\$

债券估值与利率结构 (Bond Valuation and Interest Rates)

【利率基础】
• 利率：债务工具的融资成本，借款人向贷款人支付的补偿
• 要求收益率：权益工具的资金成本，销售所有权权益获得资金的成本
• 实际利率：无通胀、无风险、无流动性偏好的均衡利率

【名义利率计算公式】
$$\text{名义利率} = \text{实际利率} + \text{通胀溢价} + \text{风险溢价}$$
r1 = r* + IP + RP1
其中：r* = 实际利率，IP = 通胀溢价，RP = 风险溢价

【债券基本特征】
• 票面利率：债券面值的年付息百分比
• 面值：债券到期时须偿还的本金金额
• 到期日：债券到期、必须偿还本金的时间

【债券估值基本模型】
B0 = I × [Σ(1/(1+rd)t)] + M × [1/(1+rd)n]
其中：B0 = 债券价值，I = 年利息，M = 面值，rd = 要求收益率，n = 到期年数

【债券价格与收益率关系】
• 折价债券：要求收益率 > 票面利率，债券价格 < 面值
• 溢价债券：要求收益率 < 票面利率，债券价格 > 面值
• 平价债券：要求收益率 = 票面利率，债券价格 = 面值

【利率风险】
• 定义：利率变化对债券价值的影响
